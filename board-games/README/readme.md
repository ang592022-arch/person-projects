This board game project is my semester assignment, divided into two parts: Code 1 and Code 2.
We implement a C++ program that can be used to build many board games, like Chinese chess (Xiang Qi), 
Western Chess, Weiqi, and Gobang.  The program can put stones on a board and print the board. 
In Code1, we built classes for a board game engine, that can set up a board, some players put 
stones on the board, and print the board. 
In Code2, we will continue to implement game logic, so that complete board games can be played.
